# Capstone Project Proposal

## Layihə adı

Unbeatable Games Platform

## Problem təsviri

İstifadəçilər üçün sadə və strateji düşüncəni inkişaf etdirən oyun sistemlərinə ehtiyac var. Mövcud mini oyunların böyük hissəsi yalnız əyləncə məqsədli qurulur və alqoritmik optimal qərarvermə tərəfi zəif olur. Bu layihədə məqsəd Python dilində hazırlanmış, bir neçə oyunu birləşdirən və optimal strategiyaları nümayiş etdirən platforma qurmaqdır.

## Məqsədlər

- Bir platformada bir neçə mini oyunu toplamaq
- Oyunlarda optimal strategiya və AI yanaşmaları tətbiq etmək
- Modular və oxunaqlı kod strukturu qurmaq
- Test və performans müqayisəsi aparmaq
- Layihəni kursun texniki tələblərinə uyğun olaraq Python dilində təqdim etmək

## İlkin oyunlar

1. `Tic Tac Toe`
   - 3x3 oyun taxtası
   - Minimax alqoritmi ilə udulmaz AI
2. `Number Target`
   - Oyunçular növbə ilə `1`-dən `N`-ə qədər ədəd seçirlər
   - Hər gedişdən sonra cəm artır
   - Məqsəd verilmiş hədəfə ilk çatmaqdır
   - Oyun riyazi optimal strategiya ilə analiz ediləcək
3. `Connect Four` (planlaşdırılan üçüncü oyun)
   - Daha böyük vəziyyət məkanı
   - Heuristik Minimax ilə qərarvermə

## İstifadə olunacaq verilən strukturları və alqoritmlər

### Verilən strukturları

- Array və List
- Matrix
- Dictionary / HashMap
- Game state ağacı
- Queue və ya stack əsaslı keçidlər üçün yardımçı strukturlar

### Alqoritmlər

- Minimax
- Heuristik qiymətləndirmə
- Riyazi optimal strategiya
- Backtracking və game tree search

## Gözlənilən nəticələr

Layihənin sonunda Python dilində işləyən, bir neçə oyunu dəstəkləyən və optimal strategiyalar təqdim edən platforma əldə ediləcək. Hər oyun üçün istifadəçi təlimatı, test ssenariləri, qısa performans analizi və seçilmiş verilən strukturlarının əsaslandırılması təqdim olunacaq.

## İş planı

- Həftə 5-7: mövzunun yekunlaşdırılması və proposal
- Həftə 7-9: platforma skeleti və Tic Tac Toe
- Həftə 9-10: Number Target oyunu
- Həftə 10-11: üçüncü oyun və progress report
- Həftə 12-14: test, analiz və UI cilalanması
- Həftə 15: final təqdimat və hesabat

## Layihənin aktuallığı

Bu layihə alqoritmlərin praktik tətbiqini göstərir, oyun nəzəriyyəsi ilə proqramlaşdırmanı birləşdirir və komanda şəklində modul sistem hazırlamaq bacarığını nümayiş etdirir.
